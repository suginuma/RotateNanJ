# RotateNanJ
やきうのお兄ちゃんであそぼう

## デモ
https://unityroom.com/games/yakiu

## 操作方法
a or LEFT ：左に転がす  
d or RIGHT：右に転がす  
ENTER     ：増やす(倍々に増えるので注意)
